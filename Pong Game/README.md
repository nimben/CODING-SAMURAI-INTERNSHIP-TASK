## How to Run the Project (Source Code)

1. Install **Unity Hub**
2. Make sure you have the correct Unity version:
   - **Unity 6.4 (6000.4.1f1)** _(or compatible version)_

3. Clone or download this repository:

   ```bash
   git clone https://github.com/nimben/CODING-SAMURAI-INTERNSHIP-TASK
   ```

4. Open Unity Hub → Click **Add Project**
5. Select the project folder
6. Open the project and press **Play**
