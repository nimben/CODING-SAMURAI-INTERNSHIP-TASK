Pong Game (Unity)
A simple 2D Pong game built using Unity as part of my game development learning and internship tasks.

How to Run the Project
Install Unity Hub

Make sure you have the correct Unity version:

Unity 6.4(6000.4.1f1) (or compatible version)
Clone or download this repository: git clone https://github.com/nimben/CODING-SAMURAI-INTERNSHIP-TASK

Open Unity Hub → Click Add Project

Select the project folder

Open the project and press ▶️ Play

## Play the Game

Click below to download and play:

[Download Pong Game](https://github.com/nimben/CODING-SAMURAI-INTERNSHIP-TASK/releases/latest)

1. Download the ZIP  
2. Extract it  
3. Run PongGame.exe
